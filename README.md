# ORMY

ORMY - Ok Really Maybe Yes

This repository provides bunch of small abstractions for examples on how to model complex modular systems. It's intentionally overcomplicated.

# Trying it out

Using sbt:

```bash
sbt
akkaApp/run
```
or 
```bash
sbt 
http4sApp/run
```
